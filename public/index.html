PDF Generation Service

Welcome to the PDF Generation Service. This service allows you to generate PDF reports of tasks from the BauBuddy API.

API Endpoints

GET /service

Generate a PDF of tasks from the BauBuddy API.

Parameters:

  • username - Your BauBuddy username
  • password - Your BauBuddy password

Example:

/service?username=365&password=1

GET /health

Check the health of the service.

Example:

/health

GET /api/docs

Interactive API documentation with Swagger UI.

Example:

/api/docs
